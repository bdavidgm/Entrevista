## Concepto
Box apila a sus hijos unos sobre otros (como capas en el eje Z), alineándolos con contentAlignment. Sirve para superponer elementos: texto encima de una [imagen](pregunta:282), un badge sobre un [icono](pregunta:283) o un indicador de carga centrado.

Ejemplo:
Box(
    modifier = Modifier.size(120.dp),
    contentAlignment = Alignment.Center
) {
    Image(painter = painter, contentDescription = null)
    CircularProgressIndicator()
}

ExplicaciónKotlin:
Box(...) usa el scope BoxScope, donde cada hijo puede posicionarse con Modifier.align(...). contentAlignment = Alignment.Center centra por defecto a los hijos que no definan su propia alineación. Los hijos se dibujan en orden de declaración: primero Image (al fondo) y luego CircularProgressIndicator (encima). Es el equivalente conceptual a FrameLayout en el sistema de [Views](pregunta:39).

Consejo:
El orden de declaración define el eje Z: lo último queda arriba. Usa Modifier.align dentro del Box para posicionar hijos concretos.
