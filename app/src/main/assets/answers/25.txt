Concepto:
El operador :: crea una referencia a un símbolo (clase, función, propiedad o constructor) en lugar de invocarlo. Se usa para pasar funciones como valores (function references), obtener KClass (MyClass::class) o referenciar propiedades para reflexión.

Las referencias a funciones encajan donde se espera un tipo función (Lambda), pero suelen ser más legibles y reutilizables.

Ejemplo:
fun isOdd(n: Int) = n % 2 != 0

val odds = listOf(1, 2, 3).filter(::isOdd)

val classRef: KClass<User> = User::class
val nameProp = User::name

users.sortedBy(User::name)
button.setOnClickListener(::onButtonClick)

Consejo:
"::funcion = function reference; Tipo::class = KClass". Para acceder a Class<T> Java usa Tipo::class.java.
