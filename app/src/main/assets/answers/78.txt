Concepto:
Una coroutine es una unidad de trabajo ligera y suspendible que Kotlin ejecuta de forma cooperativa, permitiendo código asíncrono secuencial sin bloquear hilos. En Android sustituye callbacks anidados y simplifica operaciones de red y BD junto con ViewModelScope.

Explicación detallada:
Las coroutines se programan en dispatchers (Main, IO, Default). Una función suspend puede pausarse sin bloquear el hilo del sistema; al reanudarse continúa. Miles de coroutines pueden coexistir frente a pocos threads.

CoroutineScope define ciclo de vida: viewModelScope, lifecycleScope, applicationScope. launch inicia trabajo fire-and-forget; async devuelve Deferred con resultado. Structured concurrency: los scopes padre cancelan hijos.

Integración Android: Main dispatcher actualiza UI tras withContext(IO). Retrofit y Room exponen suspend. Flow para streams múltiples de valores.

Errores: CoroutineExceptionHandler en scope raíz; try/catch en suspend. No uses GlobalScope en producción (sin lifecycle). runBlocking solo en tests o main bloqueado puntual.

En el contexto de proyectos Android reales, combina esta idea con las guías oficiales de Android Developers y con métricas de calidad (crashes, ANR, Vitals) cuando aplique. En code review, pregunta siempre por ciclo de vida, testabilidad y si la solución escala cuando el equipo crece.

Mencionar versiones recientes de API o biblioteca (Material3, Paging 3, Hilt 2.x) demuestra que sigues el ecosistema, sin necesidad de memorizar números de versión exactos.

Cuándo usarlo:
Toda operación async en Kotlin moderno: red, BD, combinación de fuentes, debounce en ViewModel. Callback APIs legacy: suspendCoroutine o adaptadores.

Ejemplo:
viewModelScope.launch {
    _uiState.update { it.copy(loading = true) }
    val data = withContext(Dispatchers.IO) { repository.fetch() }
    _uiState.update { it.copy(loading = false, items = data) }
}

Consejo para entrevista:
Explica suspend, scope y dispatchers. No digas que coroutine es un hilo. Seguimiento: diferencia thread vs coroutine — muchas coroutines, pocos threads.
