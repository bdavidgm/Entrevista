## Concepto
Lottie renderiza animaciones vectoriales exportadas desde After Effects (archivos `.json` / `.lottie`). En Compose se usa `LottieAnimation` con `rememberLottieComposition`.

Ejemplo:
val composition by rememberLottieComposition(
    LottieCompositionSpec.RawRes(R.raw.loading)
)
val progress by animateLottieCompositionAsState(
    composition = composition,
    iterations = LottieConstants.IterateForever
)

LottieAnimation(
    composition = composition,
    progress = { progress },
    modifier = Modifier.size(120.dp)
)

ExplicaciónKotlin:
`rememberLottieComposition` carga y [remember](pregunta:48) la composición. `animateLottieCompositionAsState` anima el progreso (aquí en bucle infinito). `LottieAnimation` es el [Composable](pregunta:46) que dibuja. También existe `LottieAnimationView` para el sistema de [Views](pregunta:39).

Consejo:
Optimiza el JSON (menos capas) para no penalizar el arranque; cachea composiciones reutilizadas.
