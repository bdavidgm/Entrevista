## Concepto
SQLDelight genera APIs Kotlin type-safe a partir de SQL. Encaja muy bien en multiplataforma y ofrece queries verificadas en compile time.

Ejemplo:
-- user.sq
selectAll:
SELECT * FROM user;

val users = userQueries.selectAll().executeAsList()

ExplicaciónKotlin:
El archivo `.sq` declara la query `selectAll`. El plugin genera `userQueries`. `executeAsList()` materializa filas tipadas en Kotlin.
