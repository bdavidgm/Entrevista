## Concepto
`produceState` crea un `State` alimentado por una [corrutina](pregunta:78) productora. `snapshotFlow` observa lecturas de estado Compose y emite un [Flow](pregunta:84) cuando cambian, útil para puente Compose → [Flow](pregunta:84).

Ejemplo:
val profile by produceState<Profile?>(null, userId) {
    value = repo.load(userId)
}
LaunchedEffect(listState) {
    snapshotFlow { listState.firstVisibleItemIndex }.collect { ... }
}

ExplicaciónKotlin:
`produceState` lanza el bloque al entrar en composición y actualiza `value`. `snapshotFlow { ... }` lee estado snapshot-aware y emite al collector en `LaunchedEffect`.

Consejo:
No los uses si basta `collectAsState` desde un [Flow](pregunta:84)/[StateFlow](pregunta:86) del [ViewModel](pregunta:68).
