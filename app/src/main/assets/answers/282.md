## Concepto
Image dibuja un gráfico: un recurso local, un vector o un bitmap. Recibe un painter (o ImageVector/ImageBitmap), un contentDescription para accesibilidad y un contentScale que define cómo se ajusta al espacio disponible.

Para imágenes de red se usan librerías como [Coil](pregunta:259) con rememberAsyncImagePainter.

Ejemplo:
Image(
    painter = painterResource(R.drawable.logo),
    contentDescription = "Logo",
    contentScale = ContentScale.Crop,
    modifier = Modifier.size(96.dp)
)

ExplicaciónKotlin:
painterResource(R.drawable.logo) crea un Painter a partir de un recurso; es una función [composable](pregunta:46). contentDescription = "Logo" da texto para lectores de pantalla (pon null si la imagen es decorativa). contentScale = ContentScale.Crop recorta para llenar el espacio manteniendo la proporción. modifier = Modifier.size(96.dp) fija el tamaño mediante el [Modifier](pregunta:51).

Consejo:
Pon un contentDescription útil, o null si es puramente decorativa. Para imágenes remotas usa [Coil](pregunta:259) en lugar de cargar bitmaps a mano.
