## Concepto
kotlinx.serialization es el framework oficial de Kotlin para serializar clases a JSON (y otros formatos) con plugins de compilador, sin reflexión en runtime en el camino habitual.

Ejemplo:
@Serializable
data class User(val id: Int, val name: String)

val json = Json.encodeToString(User(1, "Ana"))
val user = Json.decodeFromString<User>(json)

ExplicaciónKotlin:
`@Serializable` marca la [data class](pregunta:3) para el plugin. `Json.encodeToString` serializa; `decodeFromString<User>` deserializa con tipo [reified](pregunta:23). El formato por defecto es JSON.

Consejo:
Ideal en KMP y con [Ktor](pregunta:270); en Android puro también compite con [Moshi](pregunta:262)/[Gson](pregunta:261).
