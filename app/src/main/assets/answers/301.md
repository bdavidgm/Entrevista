## Concepto
kotlinx.serialization es la librería oficial de Kotlin para serializar datos (JSON, ProtoBuf, etc.) sin reflexión Java. Usa `@Serializable` y plugins de compilador; encaja bien con [Ktor](pregunta:270) y [Retrofit](pregunta:207) (con converter).

Ejemplo:
@Serializable
data class NoteDto(
    val id: String,
    val title: String,
    val body: String? = null
)

val json = Json { ignoreUnknownKeys = true }
val dto = json.decodeFromString<NoteDto>(jsonString)
val encoded = json.encodeToString(dto)

ExplicaciónKotlin:
`@Serializable` marca la [data class](pregunta:3) para el plugin. `Json { ignoreUnknownKeys = true }` configura el formato con un DSL. `decodeFromString<NoteDto>` usa un tipo [reified](pregunta:23) implícito vía función inline. `encodeToString` genera el JSON. No necesita `::class.java`. Ver [kotlinx.serialization](pregunta:263).

Consejo:
Ideal en multiplataforma (KMP). Activa el plugin `kotlinx-serialization` en Gradle.
