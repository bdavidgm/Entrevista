Concepto:
Modifier es la API encadenable de Compose para configurar layout, dibujo, semántica e interacción de un Composable: tamaño, padding, fondo, click, scroll, animaciones, etc. Cada llamada devuelve un nuevo Modifier que envuelve al anterior, y el orden importa.

Define la "extensión" del componente sin necesidad de subclasear ni de pasar muchos parámetros.

Ejemplo:
Box(
    modifier = Modifier
        .fillMaxWidth()
        .padding(16.dp)
        .background(Color.Blue)
        .clickable { onClick() }
) {
    Text("Hola", color = Color.White)
}

Consejo:
"El orden importa": padding antes o después de background cambia el resultado. Recibe Modifier en composables reutilizables (parámetro modifier: Modifier = Modifier).
