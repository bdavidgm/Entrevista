Concepto:
Las scope functions (let, run, with, apply, also) ejecutan un bloque de código en el contexto de un objeto. Se diferencian en dos ejes: cómo se refiere al receptor (this o it) y qué devuelven (el receptor o el resultado del bloque).

Tabla rápida:
- let { it }: receptor por argumento, devuelve resultado.
- run { this }: receptor implícito, devuelve resultado.
- with(x) { this }: forma sin extension; devuelve resultado.
- apply { this }: receptor implícito, devuelve el receptor.
- also { it }: receptor por argumento, devuelve el receptor.

Ejemplo:
val intent = Intent(context, Detail::class.java).apply {
    putExtra("id", 42)
    flags = Intent.FLAG_ACTIVITY_NEW_TASK
}

val length = user?.name?.let { it.trim().length }

val message = with(StringBuilder()) {
    append("Hola ")
    append("mundo")
    toString()
}

Consejo:
Elige la scope function por intención: apply/also si quieres seguir usando el objeto; let/run si quieres transformarlo. apply para configurar, also para side effects, let para nullable-safe.
