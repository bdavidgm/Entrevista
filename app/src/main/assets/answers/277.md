## Concepto
Button es un botón de acción de [Material Design](pregunta:105). Recibe un lambda onClick y su contenido (normalmente un [Text](pregunta:276) o un [Icon](pregunta:283)) en un trailing lambda con scope RowScope. Tiene variantes según la jerarquía: Button (relleno), FilledTonalButton, ElevatedButton, OutlinedButton y [TextButton](pregunta:291).

Ejemplo:
Button(
    onClick = { viewModel.guardar() },
    enabled = !isLoading,
    modifier = Modifier.fillMaxWidth()
) {
    Text("Guardar")
}

ExplicaciónKotlin:
onClick = { ... } es la lambda que se ejecuta al pulsar. enabled = !isLoading deshabilita el botón según un booleano de estado. El contenido va en el trailing lambda ({ Text("Guardar") }), que corre dentro de un RowScope; por eso puedes poner un Icon y un Text en fila. modifier = Modifier.fillMaxWidth() aplica el [Modifier](pregunta:51) para ocupar todo el ancho.

Consejo:
Elige la variante según la jerarquía: Button para la acción principal, OutlinedButton o TextButton para secundarias. Deshabilita con enabled durante cargas para evitar doble envío.
