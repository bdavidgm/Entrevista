Concepto:
mutableStateOf crea un State<T> observable por Compose: cada vez que se asigna a su .value, todas las composables que lo lean se invalidan y se programan para recomposición. Es la base del modelo reactivo de Compose.

Para tipos primitivos existen variantes optimizadas como mutableIntStateOf, mutableLongStateOf y mutableFloatStateOf que evitan autoboxing.

Ejemplo:
@Composable
fun NameInput() {
    var name by remember { mutableStateOf("") }
    TextField(value = name, onValueChange = { name = it })
    Text("Hola, $name")
}

Consejo:
"mutableStateOf = State observable de Compose; combínalo con remember o rememberSaveable". Para Int/Float, prefiere mutableIntStateOf y compañía.
