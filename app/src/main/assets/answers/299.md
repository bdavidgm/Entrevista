## Concepto
Moshi es una librería de Square para serializar/deserializar JSON a clases Kotlin. Combina bien con [Retrofit](pregunta:207) mediante `MoshiConverterFactory` y soporta [data class](pregunta:3) con el codegen de Kotlin.

Ejemplo:
@JsonClass(generateAdapter = true)
data class NoteDto(
    @Json(name = "note_id") val id: String,
    val title: String,
    val body: String? = null
)

val moshi = Moshi.Builder()
    .add(KotlinJsonAdapterFactory())
    .build()

val adapter = moshi.adapter(NoteDto::class.java)
val dto = adapter.fromJson(jsonString)
val json = adapter.toJson(dto)

ExplicaciónKotlin:
`@JsonClass(generateAdapter = true)` genera un adapter tipado. `@Json(name = "note_id")` mapea el nombre JSON al campo Kotlin. `KotlinJsonAdapterFactory` habilita defaults y nullability de Kotlin. `adapter.fromJson` / `toJson` convierten en ambos sentidos. En Retrofit: `.addConverterFactory(MoshiConverterFactory.create(moshi))`. Ver [Moshi](pregunta:262).

Consejo:
Prefiere codegen (`generateAdapter = true`) frente a reflexión pura por rendimiento y seguridad.
