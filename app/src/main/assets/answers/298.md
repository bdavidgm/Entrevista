## Concepto
Glide es una librería clásica de carga de imágenes muy usada en apps con Views. Descarga, cachea y muestra bitmaps con una API fluida (`Glide.with(...).load(...).into(...)`).

Ejemplo:
Glide.with(imageView)
    .load("https://example.com/photo.jpg")
    .placeholder(R.drawable.placeholder)
    .error(R.drawable.error)
    .circleCrop()
    .into(imageView)

// En Compose (con landscapist-glide o similar):
GlideImage(
    imageModel = { "https://example.com/photo.jpg" },
    modifier = Modifier.size(96.dp)
)

ExplicaciónKotlin:
`Glide.with(imageView)` amarra la carga al ciclo de vida del contexto/View. `.load(url)` define la fuente; `.placeholder` / `.error` muestran drawables temporales; `.circleCrop()` transforma el bitmap; `.into(imageView)` dispara la carga. En proyectos Compose modernos suele preferirse [Coil](pregunta:259). Ver [Glide](pregunta:260).

Consejo:
Usa el Context/Fragment correcto en `with(...)` para cancelar cargas al destruir la UI.
