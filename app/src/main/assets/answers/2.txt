Concepto:
El operador Elvis ?: devuelve el operando de la izquierda si no es null y, en caso contrario, evalúa y devuelve el de la derecha. Sirve para dar valores por defecto, lanzar excepciones o hacer early return de forma concisa.

Puede combinarse con throw o return porque ambos son expresiones de tipo Nothing, lo que el compilador interpreta como “si el lado izquierdo es null, sal de aquí”.

Ejemplo:
val name = user?.name ?: "Anónimo"
val id = bundle?.getString("id") ?: return
val token = prefs.getToken() ?: throw IllegalStateException("Sin sesión")

Consejo:
Diferéncialo del safe call (?.). Menciona el patrón "?: return" para salir temprano y mantener el código plano.
