Concepto:
Una propiedad delegada permite que la lógica de su getter/setter se delegue a otro objeto mediante la sintaxis val x by Delegate(). Kotlin proporciona delegados estándar como lazy, observable, vetoable, notNull y Map; también puedes implementar los tuyos definiendo getValue/setValue.

En Android es la base de patrones como by viewModels(), by activityViewModels(), by viewBinding() o by remember en Compose.

Ejemplo:
val name: String by lazy { computeExpensiveName() }

var age: Int by Delegates.observable(0) { _, old, new ->
    Log.d(TAG, "$old -> $new")
}

class Prefs(prefs: SharedPreferences) {
    var token: String? by prefs.string("token")
}

Consejo:
Distingue propiedades delegadas (by Delegate()) de delegación de clases (class A : I by impl). Menciona delegados comunes en Android.
