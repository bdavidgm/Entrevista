Concepto:
Signing firma el binario para que Play y los dispositivos verifiquen autoría e integridad. Usas un keystore (upload key) que firma el AAB; Play puede gestionar la app signing key de distribución.

V1/JAR + V2/APK + V3/v4 esquemas de firma; release debe usar configuración segura.

Ejemplo:
android {
    signingConfigs {
        create("release") {
            storeFile = file("release.keystore")
            storePassword = System.getenv("STORE_PW")
            keyAlias = "upload"
            keyPassword = System.getenv("KEY_PW")
        }
    }
}

Consejo:
Nunca subas keystore al repo en claro. Play App Signing permite rotación de upload key si pierdes acceso controlado.
