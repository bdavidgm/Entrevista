## Concepto
Icon dibuja un pictograma pequeño (normalmente de Material Icons) teñido con un color. Recibe un ImageVector (o painter), un contentDescription y un tint. Aparece dentro de botones, barras y listas.

Ejemplo:
Icon(
    imageVector = Icons.Filled.Favorite,
    contentDescription = "Favorito",
    tint = MaterialTheme.colorScheme.primary
)

ExplicaciónKotlin:
Icons.Filled.Favorite es un ImageVector del paquete de Material Icons. contentDescription describe el icono para accesibilidad (null si es decorativo). tint = MaterialTheme.colorScheme.primary aplica el color del tema; por defecto toma LocalContentColor. A menudo se combina con IconButton, que lo hace clicable y añade el área táctil y el efecto ripple.

Consejo:
Añade la dependencia material-icons-extended para el set completo de iconos. Usa el tint del tema y un contentDescription salvo que el icono sea decorativo.
