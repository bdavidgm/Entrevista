Concepto:
when es la expresión de control de flujo de Kotlin equivalente a un switch enriquecido. Puede usarse como sentencia o como expresión (que devuelve valor) y soporta condiciones arbitrarias, rangos, tipos (is) y deconstrucción.

Si se usa como expresión sobre un tipo sealed o enum, el compilador exige exhaustividad y avisa al añadir una variante nueva, lo que ayuda a mantener el código alineado con el dominio.

Ejemplo:
fun describe(value: Any): String = when (value) {
    is Int -> "Entero $value"
    in 1..9 -> "Dígito"
    "hola" -> "Saludo"
    null -> "Vacío"
    else -> "Otro"
}

Consejo:
Subraya when como expresión exhaustiva con sealed/enum (sin else). Compáralo con switch de Java y menciona el smart cast tras is.
