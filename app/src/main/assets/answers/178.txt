Concepto:
ContentResolver es la fachada para leer/escribir datos expuestos por ContentProviders (propios o del sistema: Contacts, MediaStore, Calendar). Métodos típicos: query, insert, update, delete, openInputStream/openFileDescriptor.

Unifica acceso CRUD sobre URIs content://.

Ejemplo:
val cursor = contentResolver.query(
    MediaStore.Images.Media.EXTERNAL_CONTENT_URI,
    arrayOf(MediaStore.Images.Media.DISPLAY_NAME),
    null, null, null
)

Consejo:
Cierra cursores con use. En Android 10+ el almacenamiento con scopes puede requerir SAF o permisos de medios específicos según el tipo de archivo.
