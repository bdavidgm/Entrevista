## Concepto
Room es la capa de persistencia oficial de Jetpack sobre SQLite. Genera el código de acceso a partir de anotaciones (@Entity, [@Dao](pregunta:71), @Database) y aporta verificación de queries SQL en tiempo de compilación, soporte para [coroutines](pregunta:78)/[Flow](pregunta:84) y migraciones explícitas.

Sus tres bloques son: Entity (tabla), DAO (operaciones tipadas) y Database (singleton que las une y configura las migraciones).

Ejemplo:
@Entity
data class User(@PrimaryKey val id: String, val name: String)

@Dao
interface UserDao {
    @Query("SELECT * FROM User") fun all(): Flow<List<User>>
    @Insert(onConflict = OnConflictStrategy.REPLACE) suspend fun upsert(user: User)
}

@Database(entities = [User::class], version = 1)
abstract class AppDb : RoomDatabase() { abstract fun userDao(): UserDao }

ExplicaciónKotlin:
@Entity [data class](pregunta:3) User(@PrimaryKey val id: String, val name: String) es [data class](pregunta:3) con anotación Room; @PrimaryKey marca id; propiedades val inmutables definen columnas. [@Dao](pregunta:71) interface UserDao declara contrato que Room implementa en compile-time. @Query("SELECT * FROM User") fun all(): [Flow](pregunta:84)<List<User>> devuelve [Flow](pregunta:84) reactivo, no [suspend](pregunta:79), porque emite múltiples veces. @Insert(onConflict = OnConflictStrategy.REPLACE) [suspend](pregunta:79) fun upsert(user: User) es [suspend](pregunta:79) con estrategia enum REPLACE. @Database(entities = [User::class], version = 1) abstract class AppDb : RoomDatabase() usa array literal [User::class] con class reference [KClass](pregunta:256) y abstract fun userDao(): UserDao sin cuerpo. Herencia : RoomDatabase() y abstract indican clase base generada por Room. Combina anotaciones, [data class](pregunta:3), interfaces, generics [Flow](pregunta:84)<List<User>> y [corrutinas](pregunta:78) [suspend](pregunta:79).

Consejo:
"Queries validadas en compilación + [Flow](pregunta:84) reactivo". Para producción cuida las migraciones; nunca uses fallbackToDestructiveMigration en release.
