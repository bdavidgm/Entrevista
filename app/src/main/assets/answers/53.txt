Concepto:
LaunchedEffect es el side effect estándar de Compose para lanzar trabajo suspendido durante la composition. Crea un CoroutineScope ligado a la composición; cuando alguna de sus keys cambia, cancela la coroutine en curso y lanza otra nueva. Al salir el Composable, también se cancela.

Sirve para iniciar trabajo asíncrono que depende del ciclo de vida del Composable: colectar Flows, lanzar animaciones, hacer requests on-screen.

Ejemplo:
@Composable
fun UserScreen(userId: String, viewModel: UserViewModel = hiltViewModel()) {
    LaunchedEffect(userId) {
        viewModel.load(userId)
    }
    val user by viewModel.user.collectAsStateWithLifecycle()
    UserContent(user)
}

Consejo:
"Las keys controlan cuándo reinicia". Si no debe reiniciar nunca, usa Unit como key. Si depende de algo que cambia, pásalo como key.
