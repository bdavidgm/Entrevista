## Concepto
TextField (y su variante OutlinedTextField) es el campo de entrada de texto de Compose. Es un componente controlado: recibe el value actual y notifica cambios con onValueChange; tú guardas ese valor en estado con [mutableStateOf](pregunta:49) y [remember](pregunta:48).

Ejemplo:
var nombre by remember { mutableStateOf("") }

OutlinedTextField(
    value = nombre,
    onValueChange = { nombre = it },
    label = { Text("Nombre") },
    singleLine = true
)

ExplicaciónKotlin:
var nombre by remember { mutableStateOf("") } declara un estado que sobrevive a las [recomposiciones](pregunta:47); by es delegación de propiedad. value = nombre muestra el estado actual y onValueChange = { nombre = it } lo actualiza con el nuevo texto (it), lo que dispara recomposición y refleja el cambio en pantalla. label es un [composable](pregunta:46) opcional. Este ciclo value/onValueChange es el patrón de [state hoisting](pregunta:50): el TextField no guarda su propio texto.

Consejo:
Nunca ignores onValueChange o el campo parecerá "congelado". Usa keyboardOptions y visualTransformation para email, números o contraseñas.
