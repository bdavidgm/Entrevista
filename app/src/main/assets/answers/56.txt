Concepto:
derivedStateOf crea un State cuyo valor se calcula a partir de otros States y se "memoriza": el Composable lector solo se recompone si el resultado derivado realmente cambia, aunque las fuentes cambien con frecuencia. Es la receta para evitar recomposiciones excesivas en valores derivados.

Caso típico: mostrar un botón solo cuando el scroll ha pasado un umbral, sin recomponer en cada píxel.

Ejemplo:
val listState = rememberLazyListState()
val showFab by remember {
    derivedStateOf { listState.firstVisibleItemIndex > 0 }
}

if (showFab) ScrollToTopFab { /* ... */ }

Consejo:
"derivedStateOf cuando la fuente cambia mucho pero el resultado pocas veces". Sin él, leer firstVisibleItemIndex causaría recomposiciones por cada frame de scroll.
