## Concepto
Moshi es una librería de Square para JSON, moderna y amigable con Kotlin (codegen o reflexión), a menudo usada con [Retrofit](pregunta:207).

Ejemplo:
val moshi = Moshi.Builder().build()
val adapter = moshi.adapter(User::class.java)
val user = adapter.fromJson(json)

ExplicaciónKotlin:
`Moshi.Builder().build()` crea el motor. `adapter(User::class.java)` obtiene un `JsonAdapter`. `fromJson` parsea el texto. Con Kotlin codegen se generan adapters en compile time.

Consejo:
Contrasta con [Gson](pregunta:261) (reflexión) y [kotlinx.serialization](pregunta:263) (multiplataforma, oficial Kotlin).
