Concepto:
PowerManager ofrece flags y control de energía: isInteractive, newWakeLock para mantener CPU/pantalla encendida de forma explícita. WakeLock debe adquirirse por intervalos cortos y liberarse en finally; abuso drena batería y molesta al usuario.

Doze y App Standby limitan wakeups aunque tengas locks.

Ejemplo:
val wakeLock = powerManager.newWakeLock(
    PowerManager.PARTIAL_WAKE_LOCK, "MyApp::SyncTag"
)
wakeLock.acquire(10_000L)
try { sync() } finally { if (wakeLock.isHeld) wakeLock.release() }

Consejo:
Prefiere WorkManager + constraints antes que PARTIAL_WAKE_LOCK. Para mantener pantalla encendida en video, FLAG_KEEP_SCREEN_ON en la ventana es más claro.
