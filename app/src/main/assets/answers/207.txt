Concepto:
Retrofit convierte una interfaz HTTP en cliente tipado mediante anotaciones @GET, @POST, @Body, @Query. Se apoya en OkHttp para conexión, interceptores y timeouts, y en un converter (Moshi/Gson) para JSON.

Suspend functions o Call/Flow integran bien con corrutinas.

Ejemplo:
interface Api {
    @GET("users/{id}")
    suspend fun user(@Path("id") id: String): UserDto
}

val retrofit = Retrofit.Builder()
    .baseUrl("https://api.example.com/")
    .addConverterFactory(MoshiConverterFactory.create())
    .client(okHttp)
    .build()
    .create(Api::class.java)

Consejo:
Centraliza auth, logging y reintentos en interceptores OkHttp. Usa Response<T> si necesitas leer códigos HTTP sin excepción.
