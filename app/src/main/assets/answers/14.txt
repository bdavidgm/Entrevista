Concepto:
Las raw strings se delimitan con triples comillas (""") y permiten escribir cadenas multilínea sin necesidad de escapar caracteres como \n o \", preservando saltos de línea y comillas tal cual aparecen.

Combinadas con trimIndent() o trimMargin() se eliminan los espacios de la indentación común, manteniendo el código legible.

Ejemplo:
val json = """
    {
      "id": 1,
      "name": "Ana"
    }
""".trimIndent()

val query = """
    |SELECT *
    |FROM users
    |WHERE active = 1
""".trimMargin()

Consejo:
Útiles para JSON, SQL y textos largos. Menciona trimIndent / trimMargin para limpiar la sangría sin afectar al contenido.
