Concepto:
Animatable es una API de Compose para animar un valor de forma imperativa dentro de una coroutine, ofreciendo control sobre el ciclo de la animación (animateTo, snapTo, stop). Es complemento de las APIs declarativas como animate*AsState y útil cuando necesitas combinar gestos con animaciones.

Ejemplo:
@Composable
fun BouncyBox() {
    val offsetX = remember { Animatable(0f) }
    LaunchedEffect(Unit) {
        offsetX.animateTo(200f, animationSpec = spring())
    }
    Box(
        Modifier
            .offset { IntOffset(offsetX.value.toInt(), 0) }
            .size(48.dp)
            .background(Color.Red)
    )
}

Consejo:
"Animatable = animación imperativa controlable". Para casos declarativos simples, prefiere animate*AsState; usa Animatable cuando necesitas pausar, reiniciar o combinar con gestos.
