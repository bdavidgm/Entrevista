## Concepto
Ktor Client es un cliente HTTP multiplataforma de JetBrains, idiomático en Kotlin ([corrutinas](pregunta:78), serializers), alternativa a [Retrofit](pregunta:207)/[OkHttp](pregunta:208) en apps KMP o Kotlin-first.

Ejemplo:
val client = HttpClient(CIO) {
    install(ContentNegotiation) { json() }
}
val user: User = client.get("https://api.example.com/user").body()

ExplicaciónKotlin:
`HttpClient(CIO)` crea el cliente con engine CIO. `install(ContentNegotiation) { json() }` negocia JSON (p. ej. [kotlinx.serialization](pregunta:263)). `get(...).body()` es [suspend](pregunta:79) y deserializa a `User`.

Consejo:
Compara con [Retrofit](pregunta:207): Ktor brilla en KMP; [Retrofit](pregunta:207) sigue muy presente en Android clásico.
