Concepto:
Para saber si hay red usable se usa ConnectivityManager: activeNetwork + getNetworkCapabilities o, mejor, registerDefaultNetworkCallback para recibir onAvailable/onLost/onCapabilitiesChanged. NET_CAPABILITY_VALIDATED indica Internet real (no solo Wi‑Fi asociado sin portal).

El permiso ACCESS_NETWORK_STATE es normal (manifest).

Ejemplo:
val cm = getSystemService(ConnectivityManager::class.java)
val online = cm.activeNetwork?.let { net ->
    cm.getNetworkCapabilities(net)
        ?.hasCapability(NetworkCapabilities.NET_CAPABILITY_INTERNET) == true
} == true

Consejo:
No uses APIs deprecadas tipo NetworkInfo. Para políticas de sync, WorkManager Constraints.setRequiredNetworkType es la integración natural.
