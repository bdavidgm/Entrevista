Concepto:
La palabra clave object en Kotlin declara un singleton: una clase con una única instancia que el compilador crea de forma thread-safe y lazy al primer uso. Combina declaración de clase e instancia en una sola sentencia.

Existen tres formas: object declaration (singleton de módulo), object expression (instancia anónima al estilo "anonymous inner class") y companion object (singleton ligado a una clase, equivalente a estáticos).

Ejemplo:
object Analytics {
    fun log(event: String) = println("EVENT: $event")
}

Analytics.log("open_screen")

val listener = object : View.OnClickListener {
    override fun onClick(v: View?) { /* ... */ }
}

Consejo:
Diferencia object declaration vs companion object. Menciona que es thread-safe sin doble check manual y compárate con el patrón Singleton clásico de Java.
