Concepto:
Una data class es una clase pensada para contener datos en la que el compilador genera automáticamente equals, hashCode, toString, componentN y copy a partir de las propiedades del constructor primario. Reduce mucho el boilerplate típico de POJOs en Java.

Debe tener al menos un parámetro en el constructor primario, no puede ser abstract ni sealed, y para igualdad solo cuentan las propiedades declaradas en ese constructor.

Ejemplo:
data class User(val id: String, val name: String, val isPremium: Boolean = false)

val u1 = User("1", "Ana")
val u2 = u1.copy(isPremium = true)
val (id, name) = u1

Consejo:
Subraya el método copy() para inmutabilidad y los component functions para destructuring. Recuerda que herencia entre data classes está restringida.
