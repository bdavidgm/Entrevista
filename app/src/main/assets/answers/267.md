## Concepto
Compose UI Test (semantics) permite interactuar con nodos por texto, tags o roles, afirmar estado y simular clics/scroll en tests instrumentados o en JVM con [Robolectric](pregunta:266) según setup.

Ejemplo:
composeRule.setContent { Counter() }
composeRule.onNodeWithText("Sumar").performClick()
composeRule.onNodeWithText("Clicks: 1").assertExists()

ExplicaciónKotlin:
`setContent` monta el árbol. `onNodeWithText` localiza por semántica. `performClick` simula el gesto. `assertExists` verifica el texto tras [recomposición](pregunta:47).

Consejo:
Añade `testTag`/`contentDescription` para selectores estables en entrevista y en CI.
