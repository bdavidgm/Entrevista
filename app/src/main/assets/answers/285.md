## Concepto
Spacer es un [composable](pregunta:46) vacío que solo ocupa espacio, definido por su [Modifier](pregunta:51). Sirve para separar elementos sin usar padding, o para empujar contenido con weight.

Ejemplo:
Row {
    Text("Izquierda")
    Spacer(modifier = Modifier.weight(1f))
    Text("Derecha")
}

ExplicaciónKotlin:
Spacer(modifier = Modifier.weight(1f)) no dibuja nada: solo reserva espacio. Dentro de un [Row](pregunta:279), weight(1f) hace que el Spacer absorba todo el espacio sobrante y empuje "Derecha" al extremo. También se usa Spacer(Modifier.height(8.dp)) para una separación vertical fija dentro de una [Column](pregunta:280).

Consejo:
Para separaciones uniformes prefiere Arrangement.spacedBy(...); usa Spacer con weight cuando quieras empujar elementos hacia los extremos.
