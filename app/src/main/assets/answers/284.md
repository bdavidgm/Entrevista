## Concepto
Card es un contenedor de [Material Design](pregunta:105) con esquinas redondeadas, color de superficie y, opcionalmente, elevación o borde. Agrupa contenido relacionado en un bloque visual bien delimitado.

Ejemplo:
Card(
    shape = RoundedCornerShape(16.dp),
    elevation = CardDefaults.cardElevation(defaultElevation = 4.dp),
    modifier = Modifier.fillMaxWidth()
) {
    Column(Modifier.padding(16.dp)) {
        Text("Título")
        Text("Contenido de la tarjeta")
    }
}

ExplicaciónKotlin:
shape = RoundedCornerShape(16.dp) redondea las esquinas y elevation = CardDefaults.cardElevation(...) define la sombra; los objetos con sufijo Defaults exponen los valores estándar de Material. El contenido va en un trailing lambda con scope ColumnScope, por eso metemos dentro una [Column](pregunta:280) con padding para dar aire al contenido. Para hacerla pulsable existe la sobrecarga Card(onClick = { ... }).

Consejo:
La Card no trae padding interno: añádelo tú al contenido. Usa la variante Card(onClick = ...) para tarjetas clicables.
