## Concepto
Checkbox, Switch y RadioButton son controles de selección. Todos son componentes controlados: reciben su estado (checked/selected) y notifican el cambio con onCheckedChange/onClick, guardando el valor en estado.

- Checkbox y Switch activan o desactivan una opción (booleano).
- RadioButton elige una sola opción entre varias.

Ejemplo:
var aceptado by remember { mutableStateOf(false) }

Row(verticalAlignment = Alignment.CenterVertically) {
    Checkbox(
        checked = aceptado,
        onCheckedChange = { aceptado = it }
    )
    Text("Acepto los términos")
}

ExplicaciónKotlin:
checked = aceptado refleja el estado actual y onCheckedChange = { aceptado = it } lo actualiza con el nuevo booleano (it), disparando [recomposición](pregunta:47). Es el mismo patrón controlado del [TextField](pregunta:278): el control no guarda su estado, lo hoisteas hacia arriba ([state hoisting](pregunta:50)). Switch funciona igual con onCheckedChange; RadioButton usa selected + onClick y se agrupan para que solo uno quede activo.

Consejo:
Envuelve el control y su etiqueta en un [Row](pregunta:279) con Modifier.toggleable/selectable para ampliar el área táctil y mejorar la accesibilidad.
