Concepto:
SharedPreferences es un almacén clave-valor persistente basado en archivos XML, síncrono en su API antigua y típicamente envuelto en apply() para escritura asíncrona. Hoy Google recomienda migrar a DataStore (Preferences o Proto) por seguridad de hilos y API moderna con Flow.

Sigue apareciendo en código legado y en preferencias simples de SDK.

Ejemplo:
val prefs = getSharedPreferences("settings", MODE_PRIVATE)
prefs.edit { putBoolean("dark", true) }

Consejo:
Evita leer prefs en hot paths del main thread con get* síncronos masivos. Para datos estructurados o observables, DataStore.
