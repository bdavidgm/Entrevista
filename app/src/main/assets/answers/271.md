## Concepto
Edge-to-edge dibuja la UI detrás de status/navigation bars. `WindowInsets` describe esas áreas para aplicar padding (`systemBarsPadding`, `imePadding`) y evitar solapamientos.

Ejemplo:
enableEdgeToEdge()
Scaffold { padding ->
    Box(Modifier.padding(padding).windowInsetsPadding(WindowInsets.systemBars))
}

ExplicaciónKotlin:
`enableEdgeToEdge()` pide dibujo bajo las barras del sistema. `Scaffold` aporta paddings. `windowInsetsPadding(WindowInsets.systemBars)` respeta insets de status/nav.

Consejo:
Habla de Material3, `WindowCompat` y teclado (`ime`) en entrevistas modernas de UI.
