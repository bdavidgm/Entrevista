## Concepto
SQLDelight genera APIs tipadas de Kotlin a partir de archivos `.sq`. Es multiplataforma (Android, iOS, desktop) y alternativa a [Room](pregunta:70) cuando buscas SQL explícito y KMP.

Ejemplo:
-- notes.sq
CREATE TABLE note (
  id INTEGER NOT NULL PRIMARY KEY AUTOINCREMENT,
  title TEXT NOT NULL,
  body TEXT NOT NULL
);

selectAll:
SELECT * FROM note ORDER BY id DESC;

insert:
INSERT INTO note(title, body) VALUES (?, ?);

// Kotlin
val queries = database.noteQueries
queries.insert("Título", "Cuerpo")
val notes: Flow<List<Note>> = queries.selectAll().asFlow().mapToList(Dispatchers.IO)

ExplicaciónKotlin:
Las queries en `.sq` generan métodos tipados (`insert`, `selectAll`). `asFlow().mapToList` integra con [coroutines](pregunta:78)/[Flow](pregunta:84). El plugin Gradle genera las clases en compile-time. Ver [SQLDelight](pregunta:272).

Consejo:
Úsalo si necesitas KMP o control fino del SQL; quédate en Room si ya estás 100% en el ecosistema Jetpack Android.
