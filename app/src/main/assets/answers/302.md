## Concepto
DataStore es la librería Jetpack recomendada para preferencias y datos pequeños tipados. Sustituye a [SharedPreferences](pregunta:177): Preferences DataStore (clave-valor) y Proto DataStore (esquema tipado). Expone [Flow](pregunta:84).

Ejemplo:
val Context.settingsDataStore by preferencesDataStore(name = "settings")

val DARK_MODE = booleanPreferencesKey("dark_mode")

suspend fun setDarkMode(context: Context, enabled: Boolean) {
    context.settingsDataStore.edit { prefs ->
        prefs[DARK_MODE] = enabled
    }
}

fun darkModeFlow(context: Context): Flow<Boolean> =
    context.settingsDataStore.data.map { prefs ->
        prefs[DARK_MODE] ?: false
    }

ExplicaciónKotlin:
`preferencesDataStore(name = ...)` es una [delegated property](pregunta:24) a nivel de [Context](pregunta:127). `booleanPreferencesKey` define la clave tipada. `edit { }` es [suspend](pregunta:79) y actualiza de forma atómica. `data.map { }` transforma el [Flow](pregunta:84) de preferencias. Ver [DataStore](pregunta:73).

Consejo:
No mezcles lecturas/escrituras síncronas en el hilo principal; consume el Flow desde el [ViewModel](pregunta:68).
