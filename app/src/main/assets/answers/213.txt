Concepto:
Cloud Firestore es una base de datos de documentos en la nube en tiempo real: colecciones, documentos, consultas indexadas y listeners que emiten cambios. Las reglas de seguridad controlan lectura/escritura por usuario.

Alternativa a Realtime Database con modelo de datos más flexible.

Ejemplo:
Firebase.firestore.collection("users").document(uid).get()
    .addOnSuccessListener { doc -> println(doc.getString("name")) }

Consejo:
Diseña el esquema para consultas (índices compuestos). Offline persistence está habilitada por defecto en móvil; vigila tamaño de cache.
