Concepto:
NotificationChannel agrupa notificaciones con el mismo id y permite al usuario silenciarlas o cambiar importancia por categoría (mensajes, promos, media). Obligatorio desde API 26; sin canal, la notificación no se muestra.

Crea canales en arranque o antes de la primera notificación; los nombres visibles los ve el usuario en Ajustes.

Ejemplo:
if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.O) {
    val ch = NotificationChannel(
        "sync",
        "Sincronización",
        NotificationManager.IMPORTANCE_LOW
    ).apply { description = "Progreso de subida de archivos" }
    notificationManager.createNotificationChannel(ch)
}

Consejo:
No borres y recrees canales en cada arranque con distinto id. IMPORTANCE_LOW evita heads-up intrusivos para tareas rutinarias.
