Concepto:
Las notificaciones se construyen con NotificationCompat, se asocian a un canal (Android 8+) y se muestran con NotificationManager.notify. Desde Android 13 hace falta el permiso en runtime POST_NOTIFICATIONS.

Foreground services muestran notificación obligatoria al iniciar el servicio.

Ejemplo:
if (Build.VERSION.SDK_INT >= 33) {
    requestPermissions(arrayOf(Manifest.permission.POST_NOTIFICATIONS), 0)
}

val channel = NotificationChannel("msg", "Mensajes", NotificationManager.IMPORTANCE_DEFAULT)
nm.createNotificationChannel(channel)

val notif = NotificationCompat.Builder(this, "msg")
    .setSmallIcon(R.drawable.ic_stat)
    .setContentTitle("Nuevo mensaje")
    .setContentText("Hola")
    .build()

nm.notify(1, notif)

Consejo:
Canal con IMPORTANCE adecuada (no todo HIGH). Deep link con PendingIntent; agrupa notificaciones relacionadas con setGroup.
