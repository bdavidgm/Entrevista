Concepto:
as es el operador de cast inseguro: convierte un valor al tipo indicado y lanza ClassCastException si no encaja. as? es el cast seguro: devuelve null en lugar de lanzar excepción cuando el tipo no coincide.

as? combinado con ?: es el patrón habitual para convertir y proporcionar un fallback sin riesgos.

Ejemplo:
val any: Any = "hola"

val s1 = any as String          // ok
val s2 = any as Int             // ClassCastException
val s3 = any as? Int            // null
val s4 = (any as? Int) ?: -1    // -1

Consejo:
Preferir as? + Elvis sobre as. Mencionar smart cast como alternativa cuando ya hay un is previo.
