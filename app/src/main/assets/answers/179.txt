Concepto:
getSystemService(Class) devuelve el servicio del sistema tipado (ConnectivityManager, AlarmManager, ClipboardManager, etc.) asociado al Context. Es la forma recomendada frente a getSystemService(String) con cast manual.

Cada servicio encapsula hardware o políticas del sistema.

Ejemplo:
val cm = getSystemService(ConnectivityManager::class.java)
val nm = getSystemService(NotificationManager::class.java)
val imm = getSystemService(InputMethodManager::class.java)

Consejo:
Usa ContextCompat o applicationContext cuando el servicio no dependa de la Activity. Algunos servicios requieren API mínima: comprueba versiones.
