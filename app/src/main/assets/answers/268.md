## Concepto
WebView muestra contenido web dentro de la app. Útil para páginas híbridas, OAuth embebido o ayuda HTML, con cuidados de seguridad (JS, file access).

Ejemplo:
webView.settings.javaScriptEnabled = true
webView.loadUrl("https://example.com")

ExplicaciónKotlin:
`settings.javaScriptEnabled` activa JS (riesgo si la URL no es de confianza). `loadUrl` carga la página. Conviene `WebViewClient` para controlar navegación y SSL.

Consejo:
Menciona seguridad: nunca habilites JS en contenido no confiable; valora Custom Tabs como alternativa.
