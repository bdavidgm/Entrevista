## Concepto
Slider permite elegir un valor dentro de un rango deslizando el pulgar. Es un control controlado: recibe value (Float), notifica con onValueChange y acepta valueRange y steps.

Ejemplo:
var volumen by remember { mutableStateOf(0.5f) }

Slider(
    value = volumen,
    onValueChange = { volumen = it },
    valueRange = 0f..1f
)

ExplicaciónKotlin:
value = volumen es el valor actual (Float). onValueChange = { volumen = it } actualiza el estado mientras se arrastra. valueRange = 0f..1f define el rango con un [range](pregunta:12) de Float. steps divide el rango en tramos discretos (0 = continuo). Igual que otros controles, sigue el patrón value/onValueChange del [state hoisting](pregunta:50).

Consejo:
Usa onValueChangeFinished para reaccionar solo cuando el usuario suelta el pulgar (por ejemplo, guardar). Redondea el value si necesitas enteros.
