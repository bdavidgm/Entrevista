## Concepto
Text es el [composable](pregunta:46) para mostrar texto en pantalla. Recibe un String (o AnnotatedString) y parámetros de estilo como color, fontSize, fontWeight, textAlign, maxLines y overflow.

Se apoya en el sistema de tipografía de [Material Design](pregunta:105) a través de MaterialTheme.typography.

Ejemplo:
Text(
    text = "Hola mundo",
    style = MaterialTheme.typography.titleMedium,
    color = MaterialTheme.colorScheme.primary,
    maxLines = 1,
    overflow = TextOverflow.Ellipsis
)

ExplicaciónKotlin:
Text(...) es una llamada a un [composable](pregunta:46) con parámetros nombrados. text es el único obligatorio. style recibe un TextStyle tomado de MaterialTheme.typography (un CompositionLocal del tema). color usa el esquema de color de Material. maxLines = 1 junto con overflow = TextOverflow.Ellipsis recortan el texto con puntos suspensivos si no cabe. Los demás parámetros tienen valores por defecto, por eso solo especificamos los que necesitamos.

Consejo:
Usa los estilos de MaterialTheme.typography en vez de tamaños fijos, y combina maxLines + overflow para textos que pueden desbordar.
