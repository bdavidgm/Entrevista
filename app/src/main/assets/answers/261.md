## Concepto
Gson es una librería de Google para serializar y deserializar JSON a objetos Java/Kotlin mediante reflexión.

Ejemplo:
val user = Gson().fromJson(json, User::class.java)
val jsonOut = Gson().toJson(user)

ExplicaciónKotlin:
`Gson().fromJson` convierte el String JSON en `User` usando `User::class.java`. `toJson` hace el camino inverso. En Kotlin conviene [data classes](pregunta:3) y, si hay nullability estricta, validar campos.

Consejo:
Menciona TypeToken para listas genéricas y alternativas tipadas como [Moshi](pregunta:262) o [kotlinx.serialization](pregunta:263).
