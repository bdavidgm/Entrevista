Concepto:
El parámetro key en LazyColumn/LazyRow le dice a Compose cómo identificar cada ítem para mantener su estado interno (estado de scroll, animaciones, foco) al reordenar, insertar o eliminar elementos de la lista. Sin key se usa la posición, lo que provoca recomposiciones innecesarias y "saltos" visuales al cambiar la lista.

La key debe ser estable y única por ítem (típicamente el id del dominio).

Ejemplo:
LazyColumn {
    items(users, key = { it.id }) { user ->
        UserRow(user)
    }
}

Consejo:
"key estable por ítem evita reusar la posición". Imprescindible para animateItemPlacement y para conservar el estado de hijos editables al reordenar.
