Concepto:
BiometricPrompt es la API unificada para huella, rostro u otros métodos Class 3/2/1 según hardware. Muestra el diálogo del sistema, gestiona intents de bloqueo y sustituye a FingerprintManager deprecado.

Se configura con BiometricPrompt.PromptInfo (título, subtítulo, allowedAuthenticators).

Ejemplo:
val prompt = BiometricPrompt(this, executor, object : BiometricPrompt.AuthenticationCallback() {
    override fun onAuthenticationSucceeded(result: BiometricPrompt.AuthenticationResult) {
        unlockVault()
    }
})
prompt.authenticate(
    BiometricPrompt.PromptInfo.Builder()
        .setTitle("Desbloquear")
        .setNegativeButtonText("Cancelar")
        .build()
)

Consejo:
Ofrece fallback con credencial (setDeviceCredentialAllowed o flujo propio). allowedAuthenticators permite exigir hardware fuerte según requisitos de seguridad.
