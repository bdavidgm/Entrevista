Concepto:
DiffUtil compara dos listas y calcula un conjunto mínimo de insert/remove/move para actualizar RecyclerView sin recargar todo. ListAdapter en Kotlin delega en AsyncListDiffer para aplicar el resultado en el hilo principal.

Mejora rendimiento y animaciones de cambio.

Ejemplo:
private object UserDiff : DiffUtil.ItemCallback<User>() {
    override fun areItemsTheSame(a: User, b: User) = a.id == b.id
    override fun areContentsTheSame(a: User, b: User) = a == b
}

class UserAdapter : ListAdapter<User, VH>(UserDiff)

Consejo:
Si solo cambia un campo, implementa getChangePayload para actualizar parcialmente el ViewHolder y evitar rebind completo.
