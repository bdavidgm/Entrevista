Concepto:
La destructuring declaration permite descomponer un objeto en varias variables en una sola línea usando paréntesis: val (a, b) = pair. Funciona con cualquier tipo que defina las funciones componentN (operator fun componentN()), como las data class, Pair, Triple, Map.Entry.

Es útil para iterar mapas, devolver múltiples valores y simplificar la lectura de tuplas.

Ejemplo:
data class User(val id: String, val name: String)
val user = User("1", "Ana")
val (id, name) = user

for ((key, value) in mapOf("a" to 1, "b" to 2)) {
    println("$key=$value")
}

val (success, error) = repository.load()

Consejo:
Las variables se asignan por posición, no por nombre. Si solo necesitas algunas, usa _ para ignorar: val (_, name) = user.
