## Concepto
Los App Widgets muestran información en la pantalla de inicio. Glance es el toolkit Jetpack moderno (Compose-like) para construir widgets de forma declarativa.

Ejemplo:
class MyWidget : GlanceAppWidget() {
    override suspend fun provideGlance(context: Context, id: GlanceId) {
        provideContent { Text(text = "Hola") }
    }
}

ExplicaciónKotlin:
`GlanceAppWidget` define el widget. `provideGlance` es [suspend](pregunta:79) y llama `provideContent` con UI Glance (`Text`). El sistema actualiza según políticas de refresh.

Consejo:
Distingue widgets clásicos (RemoteViews) de Glance; menciona límites de actualización en background.
