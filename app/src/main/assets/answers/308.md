## Concepto
Apollo Kotlin es el cliente GraphQL tipado: a partir de operaciones `.graphql` genera modelos y un `ApolloClient` que ejecuta queries/mutations (sobre [OkHttp](pregunta:208)).

Ejemplo:
# GetNotes.graphql
query GetNotes {
  notes {
    id
    title
  }
}

val apollo = ApolloClient.Builder()
    .serverUrl("https://api.example.com/graphql")
    .okHttpClient(okHttpClient)
    .build()

suspend fun loadNotes(): List<GetNotesQuery.Note> {
    val response = apollo.query(GetNotesQuery()).execute()
    return response.data?.notes.orEmpty()
}

ExplicaciónKotlin:
El plugin genera `GetNotesQuery` y tipos anidados. `apollo.query(...).execute()` es [suspend](pregunta:79). `response.data?.notes` usa el [safe call](pregunta:17). Ideal cuando el backend es GraphQL en vez de REST. Ver [Apollo](pregunta:215).

Consejo:
Versiona los `.graphql` junto al código y habilita el normalized cache si necesitas offline.
