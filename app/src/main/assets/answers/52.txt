Concepto:
LazyColumn y LazyRow son los equivalentes a RecyclerView en Compose: solo componen y miden los ítems visibles en pantalla, reciclando huecos en memoria a medida que se hace scroll. Reciben los datos a través de un DSL (items, itemsIndexed, item, stickyHeader).

Para optimizar, conviene declarar una key estable por ítem para que Compose pueda mantener el estado al reordenar y minimizar recomposiciones.

Ejemplo:
LazyColumn(
    contentPadding = PaddingValues(16.dp),
    verticalArrangement = Arrangement.spacedBy(8.dp)
) {
    items(users, key = { it.id }) { user ->
        UserCard(user)
    }
}

Consejo:
"Solo compone los ítems visibles" + "siempre pon key estable". No los anides en Column con verticalScroll (rompe el reciclaje).
