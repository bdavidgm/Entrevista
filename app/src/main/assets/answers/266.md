## Concepto
Robolectric permite tests unitarios de código Android en la JVM, simulando el framework ([Context](pregunta:127), Resources, etc.) sin emulador.

Ejemplo:
@RunWith(RobolectricTestRunner::class)
class UtilsTest {
    @Test
    fun formats() {
        val ctx = ApplicationProvider.getApplicationContext<Context>()
        assertEquals("ok", formatLabel(ctx))
    }
}

ExplicaciónKotlin:
`@RunWith(RobolectricTestRunner::class)` activa el runner. `ApplicationProvider.getApplicationContext` obtiene un [Context](pregunta:127) simulado. El test corre en `test` (JVM), más rápido que instrumented.
