Concepto:
catch es un operador de Flow que intercepta excepciones lanzadas en el upstream y permite emitir valores alternativos, relanzar o completar. Solo atrapa errores que ocurren antes en la cadena; no sustituye un try/catch alrededor de collect si el lambda del collector lanza.

Es el sitio idóneo para mapear fallos de red a estados de error en la UI.

Ejemplo:
repository.observeUsers()
    .catch { e ->
        emit(emptyList()) // o log + emitAll(fallbackFlow())
    }
    .onEach { /* ... */ }

Consejo:
Si necesitas manejar errores también en el collector, combina catch upstream con try/catch en collect o usa Result.
