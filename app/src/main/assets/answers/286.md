## Concepto
DropdownMenu es el menú emergente de opciones que aparece anclado a un elemento (por ejemplo un [icono](pregunta:283) de tres puntos). Se controla con un estado booleano expanded y contiene DropdownMenuItem clicables.

Ejemplo:
var expanded by remember { mutableStateOf(false) }

Box {
    IconButton(onClick = { expanded = true }) {
        Icon(Icons.Filled.MoreVert, contentDescription = "Menú")
    }
    DropdownMenu(
        expanded = expanded,
        onDismissRequest = { expanded = false }
    ) {
        DropdownMenuItem(
            text = { Text("Editar") },
            onClick = { expanded = false; onEditar() }
        )
        DropdownMenuItem(
            text = { Text("Eliminar") },
            onClick = { expanded = false; onEliminar() }
        )
    }
}

ExplicaciónKotlin:
expanded es un estado booleano recordado con [remember](pregunta:48). El IconButton lo pone en true para abrir el menú. DropdownMenu se muestra según expanded y onDismissRequest = { expanded = false } lo cierra al tocar fuera. Cada DropdownMenuItem recibe text (un [composable](pregunta:46)) y onClick, donde primero cerramos el menú (expanded = false) y luego ejecutamos la acción. El [Box](pregunta:281) ancla el menú junto al icono.

Consejo:
Cierra siempre el menú (expanded = false) dentro de cada onClick y en onDismissRequest. Para seleccionar el valor de un campo usa ExposedDropdownMenuBox.
