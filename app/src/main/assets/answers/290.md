## Concepto
FloatingActionButton (FAB) es el botón circular flotante para la acción principal de una pantalla (crear, añadir). Se coloca en el slot floatingActionButton de un [Scaffold](pregunta:59).

Ejemplo:
Scaffold(
    floatingActionButton = {
        FloatingActionButton(onClick = onAdd) {
            Icon(Icons.Filled.Add, contentDescription = "Agregar")
        }
    }
) { innerPadding ->
    // contenido, aplicando innerPadding
}

ExplicaciónKotlin:
FloatingActionButton(onClick = ...) recibe su contenido (normalmente un [Icon](pregunta:283)) en un trailing lambda. Al ubicarlo en el slot floatingActionButton del [Scaffold](pregunta:59), este lo posiciona automáticamente en la esquina inferior derecha respetando los insets del sistema. Existe ExtendedFloatingActionButton para incluir texto junto al icono.

Consejo:
Reserva el FAB para la acción principal y única de la pantalla. Usa ExtendedFloatingActionButton cuando el icono por sí solo no sea claro.
