Concepto:
CompositionLocal es el mecanismo de Compose para propagar valores implícitamente por el árbol sin tener que pasarlos por parámetros. Se usa para cosas transversales: tema, densidad, controladores (LocalContext, LocalLifecycleOwner) o configuraciones de UI.

Hay dos variantes: compositionLocalOf (eficiente: solo recompone lectores cuando cambia) y staticCompositionLocalOf (no se observa: recompone toda la rama bajo el Provider). Crea uno propio solo cuando varios composables anidados necesitan el mismo valor y no es práctico pasarlo por parámetro.

Ejemplo:
val LocalUser = compositionLocalOf<User?> { null }

@Composable
fun App(user: User) {
    CompositionLocalProvider(LocalUser provides user) {
        ProfileButton()
    }
}

@Composable
fun ProfileButton() {
    val user = LocalUser.current
    Text(user?.name.orEmpty())
}

Consejo:
"Para datos cross-cutting, no para reemplazar parámetros". Abusar de CompositionLocal oculta dependencias y dificulta tests.
