## Concepto
AlertDialog es el diálogo modal de [Material Design](pregunta:105) para confirmar acciones o pedir una decisión. Se muestra condicionalmente según un estado booleano y define los slots title, text, confirmButton y dismissButton. Dialog es el contenedor genérico para diálogos totalmente personalizados.

Ejemplo:
var mostrar by remember { mutableStateOf(false) }

if (mostrar) {
    AlertDialog(
        onDismissRequest = { mostrar = false },
        title = { Text("¿Eliminar?") },
        text = { Text("Esta acción no se puede deshacer.") },
        confirmButton = {
            TextButton(onClick = { mostrar = false; onEliminar() }) { Text("Eliminar") }
        },
        dismissButton = {
            TextButton(onClick = { mostrar = false }) { Text("Cancelar") }
        }
    )
}

ExplicaciónKotlin:
El diálogo solo se compone cuando mostrar es true (renderizado condicional con if). onDismissRequest = { mostrar = false } se invoca al tocar fuera o pulsar atrás. title, text, confirmButton y dismissButton son slots [composable](pregunta:46); confirmButton suele ser un TextButton que ejecuta la acción y cierra el diálogo. El dismissButton aparece a la izquierda del confirmButton.

Consejo:
No olvides poner mostrar = false en onDismissRequest y en ambos botones. Para diálogos a medida usa Dialog { ... } con tu propia Surface.
