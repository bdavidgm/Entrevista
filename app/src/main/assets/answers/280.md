## Concepto
Column apila a sus hijos verticalmente, uno debajo de otro. Es el contenedor vertical básico; su equivalente horizontal es [Row](pregunta:279). Controla el reparto vertical con verticalArrangement y la alineación horizontal con horizontalAlignment.

Ejemplo:
Column(
    modifier = Modifier.padding(16.dp),
    verticalArrangement = Arrangement.spacedBy(8.dp),
    horizontalAlignment = Alignment.CenterHorizontally
) {
    Text("Título")
    Text("Subtítulo")
}

ExplicaciónKotlin:
Column(...) coloca los hijos de arriba a abajo dentro de un ColumnScope. verticalArrangement = Arrangement.spacedBy(8.dp) inserta 8dp entre cada hijo. horizontalAlignment = Alignment.CenterHorizontally los centra en el eje X. Si el contenido excede la pantalla y necesitas scroll, añade Modifier.verticalScroll(rememberScrollState()); para muchas filas de datos usa mejor [LazyColumn](pregunta:52).

Consejo:
Column compone todos sus hijos a la vez: si son muchos usa [LazyColumn](pregunta:52). Para scroll de pocos elementos, verticalScroll es suficiente.
